## Initial set up
1. To activate virtual env (macOS):
```python
python3 -m venv myenv
source myenv/bin/activate
```
2. make install

3. Create .env file and add OpenAI API keys
```text
OPENAI_API_KEY=
OPEN_AI_ORG_ID=
```

4. make execute
